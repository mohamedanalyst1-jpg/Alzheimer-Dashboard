# Alzheimer-Dashboard
This dashboard provides a clinical deep-dive into Alzheimer's diagnostic data. By separating cognitive, functional, and lifestyle markers, the tool helps identify the strongest predictors of the disease.

# Alzheimer's Clinical Analytics Dashboard

## Project Description
Interactive Alzheimer’s Clinical Analytics Dashboard built with Python, Streamlit, Pandas, and Seaborn. The dashboard analyzes patient records, diagnosis trends, cognitive assessments, behavioral indicators, and lifestyle factors through dynamic KPIs, filters, and professional visualizations.

## Run the Project

```bash
streamlit run alzheimers_dashboard.py
```

## Technologies Used
- Python
- Streamlit
- Pandas
- Seaborn
- Matplotlib
- NumPy
